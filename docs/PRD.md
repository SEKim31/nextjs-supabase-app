# 하루모아 (HaruMoa) MVP PRD

## 🎯 핵심 정보

**목적**: 혼자 하면 포기하기 쉬운 습관을 소중한 사람과 함께 인증하고 응원하며 지속 가능하게 만드는 커뮤니티형 루틴 앱
**사용자**: 습관을 함께 만들고 싶은 모든 관계 (커플, 가족, 친구/스터디 그룹 등)

---

## 🚶 사용자 여정

```
1. 랜딩 페이지
   ↓ [로그인/회원가입 버튼 클릭]

2. 로그인 페이지 / 회원가입 페이지
   ↓ [인증 성공]

3. 대시보드 (메인 화면)
   ↓ [조건 체크]

   [그룹 없음] → 그룹 생성 페이지 / 초대 링크 접속
   [그룹 있음] → 대시보드에서 활동 시작
   ↓

4. 대시보드 활동
   • [빠른 인증 버튼] → 미션 인증 페이지 → 인증 완료 → 대시보드
   • [그룹 선택] → 그룹 상세 페이지 → [미션 생성] → 대시보드
   • [통계 보기] → 통계 페이지 → 대시보드
   • [설정] → 프로필 수정 / 알림 설정
   ↓

5. 지속적 사용
   • 매일 미션 인증
   • 그룹원 응원
   • 성취도 확인
```

## ⚡ 기능 명세

### 1. MVP 핵심 기능

| ID       | 기능명             | 설명                                                      | MVP 필수 이유              | 관련 페이지                             |
| -------- | ------------------ | --------------------------------------------------------- | -------------------------- | --------------------------------------- |
| **F001** | 그룹 생성 및 관리  | 그룹 이름, 설명, 관계 유형 설정, 최대 6명                 | 서비스 핵심 컨셉 구현      | 그룹 생성 페이지, 그룹 상세 페이지      |

| **F003** | 미션 생성 및 관리  | 제목, 설명, 반복 주기, 인증 타입 설정, 그룹당 최대 5개    | 습관 형성의 핵심 단위      | 미션 생성 페이지, 미션 상세 페이지      |
| **F004** | 데일리 인증 시스템 | 사진/텍스트/체크리스트 방식 인증, 당일 취소 가능          | 사용자 참여의 핵심 행동    | 미션 인증 페이지, 대시보드              |
| **F005** | 응원 시스템        | "최고야!" 버튼 클릭, 응원 카운트 표시                     | 관계 기반 동기부여 제공    | 대시보드, 그룹 상세 페이지              |
| **F006** | 성취도 시각화      | 오늘의 현황, 주간 달성률 그래프, 월간 캘린더, 연속 스트릭 | 사용자의 지속적 동기 부여  | 대시보드, 통계 페이지                   |
